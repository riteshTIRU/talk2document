Built an AI document assistant that allows users to upload PDF files and chat with the content using natural language.

Implemented backend APIs using Django REST Framework (DRF) to handle file uploads, storage, and chat queries.

Integrated LangChain with Hugging Face sentence-transformer embeddings (all-MiniLM-L6-v2) and ChromaDB for vector storage and retrieval.

Developed a Retrieval-Augmented Generation (RAG) pipeline enabling context-aware answers directly from uploaded documents.

Deployed an efficient Llama-based local LLM for inference on CPU, optimising performance without GPU dependency.

Designed a React frontend to manage uploads and provide a chat-style interface for user interaction.
